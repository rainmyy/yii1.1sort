yii排序封装方法
Sortsyii1.1.php适用于框架Yii1.1
文件存放位置：wwwroot/protected/components
引入方法:在框架配置文件:wwwroot/protected/config/main.php中添加:
                'import'=>array(
                    'application.components.*',
                 )
Sortsyii2.0.php适用于框架Yii2.0
文件存放位置：/common/；
引入方法：需要适用对应的命名空间: use common\library\Sorts;
